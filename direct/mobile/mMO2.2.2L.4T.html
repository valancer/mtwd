데이터 설정

설정 > 연결 > 설정 > 대상 > 옵션 > 새 대상

  • 1. 대상 이름에 “SKT 3G” 입력 후 아무 아이콘 선택
  • 2. “SKT 3G” 선택 후 “옵션” → 새 엑세스 지점
  • 3. “사용 가능한 엑세스 지점을 자동으로 확인하시겠습니까?”에서 “아니오” 선택
  • 4. “패킷 데이터” 선택 후 패킷 데이터 엑세스 지점 이름에 “web.sktelecom.com” 입력 후 “OK” 선택

MMS 사용을 위한 세팅

설정 > 연결 > 설정 > 대상 > 옵션 > 새 대상

  • 1. 대상 이름에 “SKT MMS” 입력 후 아무 아이콘 선택
  • 2. “SKT MMS” 선택 후 “옵션” → 새 엑세스 지점
  • 3. “사용 가능한 엑세스 지점을 자동으로 확인하시겠습니까?”에서 “아니오” 선택
  • 4. “패킷 데이터” 선택 후 패킷 데이터 엑세스 지점 이름에 “web.sktelecom.com” 입력 후 “OK” 선택
  • 5. “홈페이지 정보 수정”에서 “http://omms.nate.com:9082/oma_mms” 입력 후 “옵션” 클릭
  • 6. 프록시 서버 주소에 “smart.nate.com” 입력 (포트 번호 : 9093)
  • 7. MMS APN 생성 후 메뉴 → 메뉴 → 옵션 → 설정 → 멀티미디어 메시지 → 사용 엑세스 지점에서 “web.sktelecom.com” 선택
